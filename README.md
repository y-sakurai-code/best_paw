# Best Paw(ベストパウ)
## サイト概要
### サイトテーマ
愛犬家同士で犬用グッズの正直な評価を共有し、愛犬に最適な商品選びを支援する犬専用レビューサイト

### テーマを選んだ理由
私自身、犬を飼育しているのですが、ドッグフードやおもちゃ、ケア用品を選ぶ際に、広告情報だけでは自分の愛犬（犬種や年齢、体質）に本当に合うのか判断が難しいという課題を感じていました。  
特に犬は種類によって体格や噛む力、アレルギーの出やすさが大きく異なるため、実際に同じ犬種を飼っている他の飼い主の「生の声」を知るニーズは非常に高いと考えています。  
このアプリは、飼い主目線の詳細なレビューを一元管理し、ユーザーが自分の愛犬に最適なアイテムを効率的に見つけ、愛犬の健康と生活の質の向上に貢献するために制作しました。

### ターゲットユーザ
* 初めて犬を飼い始めたばかりで、どのようなドッグフードや用品を選べば良いか迷っている人
* 現在使用しているグッズに不満があり、愛犬の好みに合う新しい商品を検討している飼い主
* 特定の犬種や年齢層に特化した商品の、メリット・デメリットを詳しく知りたい人

### 主な利用シーン
* ドッグフードを切り替える前に、他の飼い主の「食いつき」や「体調の変化」を確認したい時
* 壊れにくいおもちゃや、しつけ用のおやつを探している際に、実際の使用感を確認したい時
* 特定の犬種特有の悩み（毛並み、関節ケアなど）に対する、おすすめのサプリメントやケア用品を比較したい時
